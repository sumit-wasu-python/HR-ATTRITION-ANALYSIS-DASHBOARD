# HR-ATTRITION-ANALYSIS-DASHBOARD
Interactive HR Attrition Analytics Dashboard built using Power BI, DAX, and IBM HR Analytics Dataset. Provides insights into employee attrition, job satisfaction, demographics, and workforce trends.

Project Overview
This project presents an interactive HR Analytics Dashboard built using Power BI to analyze employee attrition trends and identify key factors affecting workforce retention.
The dashboard provides insights into employee demographics, job satisfaction, salary distribution, education background, and department-wise attrition to support data-driven HR decision-making.

Business Problem
Employee attrition can significantly impact organizational productivity and recruitment costs. This dashboard helps HR teams understand:
Which departments experience the highest attrition
Which employee groups are most likely to leave
The relationship between job satisfaction and attrition
Attrition trends across age groups, gender, and education fields

Key Insights
1. Department-wise Attrition
Sales and Research departments contribute significantly to employee attrition.
2. Age Group Analysis
Employees aged 26–35 show the highest attrition rate.
3. Education Analysis
Attrition varies across education backgrounds, highlighting potential skill-retention challenges.
4. Job Satisfaction Impact
Employees with lower job satisfaction are more likely to leave the organization.
